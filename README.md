El Pollo Loco 🌵 <br> <br>
Ein browserbasiertes 2D-Jump-and-Run-Spiel, entwickelt mit modernem JavaScript und HTML5.

[➔ Hier spielen https://el-pollo-loco.maximilian-bese.de/] <br> <br>
🎮 Features <br>
Full Gameplay: Kämpfe gegen Hühner, sammle Münzen & Flaschen und besiege den Boss.

OOP-Architektur: Konsequente Nutzung von Klassen und Vererbung.

Physik: Integrierte Gravitation und Kollisionslogik.

Media: Flüssige Animationen und atmosphärische Sound-Effekte. <br> <br>

🛠 Tech Stack
Frontend: JavaScript (ES6), HTML5, CSS3 <br> <br>

🕹 Steuerung
Laufen: Pfeiltasten / A & D 

Springen: Leertaste / Pfeil oben

Werfen: Taste D <br><br>

🚀 Schnellstart
Repository klonen.

index.html im Browser öffnen.

Spaß haben! <br> <br>
